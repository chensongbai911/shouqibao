# 音频资源说明

## 需要准备的音频

### 打击音效
- **hit.mp3** - 基础打击音效
  - 时长：< 1秒
  - 格式：MP3 (推荐) 或 AAC
  - 文件大小：< 50KB
  - 音效类型：拳头打击、啪啪声、敲击声

## 音效获取渠道

### 免费音效库
1. **爱给网** (https://www.aigei.com/sound/)
   - 搜索关键词：打击、punch、hit、slap

2. **Freesound** (https://freesound.org/)
   - 搜索：punch sound, hit sound, slap sound

3. **Pixabay** (https://pixabay.com/sound-effects/)
   - 完全免费，无需署名

### 自制音效
- 使用 Audacity 软件录制和编辑
- 手机录音后剪辑

## 音频规格要求
- 格式：MP3（推荐）或 AAC
- 采样率：44100Hz
- 比特率：128kbps
- 时长：0.5-1秒
- 文件大小：< 50KB

## 临时方案
如果暂时没有音频文件，可以：
1. 暂时注释掉音效代码
2. 使用系统提示音（wx.showToast）
3. 只保留震动反馈

## 音频压缩工具
- Audacity (免费开源)
- 在线压缩：https://www.media.io/audio-compressor.html
