# 图片资源说明

## 需要准备的图片

### 受气包图片
- **bag_normal.png** - 受气包默认表情（512x512px，PNG透明底）
  - 推荐使用 AI 绘画工具生成（Midjourney/Stable Diffusion）
  - 画风：丑萌、手绘、涂鸦感
  - 表情：微笑或挑衅

- **bag_hit.png** - 受气包受击表情（512x512px，PNG透明底）
  - 表情：痛苦、惊讶
  - 可选，如不提供则使用动画效果

### 临时占位图片
如果暂时没有图片资源，可以使用以下方式：
1. 使用纯色占位符
2. 使用 emoji 表情
3. 从免费图库下载：
   - Flaticon.com
   - 爱给网 aigei.com

## 图片规格要求
- 格式：PNG（需要透明背景）
- 尺寸：512x512px 或更小
- 文件大小：< 200KB（压缩后）
- 命名规范：小写字母+下划线

## 压缩工具推荐
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
