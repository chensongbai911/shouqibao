## 🚀 快速步骤

### 1️⃣ 打开云开发控制台

```
网址: https://console.cloud.tencent.com/tcb
环境: cloud1-0g29mlsv3d4ca637
菜单: 数据库
```

---

### 2️⃣ 创建3个集合

#### 集合 1: user_data

```
名称: user_data
点击: 新增集合
输入: user_data
确定: 完成
```

#### 集合 2: achievements

```
名称: achievements
点击: 新增集合
输入: achievements
确定: 完成
```

#### 集合 3: leaderboard

```
名称: leaderboard
点击: 新增集合
输入: leaderboard
确定: 完成
```

---

### 3️⃣ 设置权限 (可选但推荐)

对每个集合，点击权限设置：

```
创建: 仅创建者
读取: 所有用户
更新: 仅创建者
删除: 仅创建者
```
