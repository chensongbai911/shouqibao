# 音频资源说明

本目录用于存放游戏音效文件。

## 所需音效文件

请在此目录下放置以下音效文件：

1. **punch.mp3** - 拳头击打音效
   - 建议：短促的打击声（约 0.2-0.5 秒）
   - 参考：拳击手套击打沙袋的声音

2. **slap.mp3** - 拖鞋拍打音效
   - 建议：清脆的拍打声（约 0.3 秒）
   - 参考：拖鞋拍打的"啪"声

3. **pan.mp3** - 平底锅敲击音效
   - 建议：金属碰撞声（约 0.4 秒）
   - 参考：锅铲敲打平底锅的"铛铛"声

## 音效要求

- **格式**: MP3（推荐）或 WAV
- **大小**: 每个文件 < 100KB（小程序包大小限制）
- **时长**: 0.2-0.5 秒（保证高频点击不卡顿）
- **音量**: 适中，建议在代码中设置 volume = 0.8

## 音效来源推荐

- [Freesound](https://freesound.org/) - 免费音效库
- [Zapsplat](https://www.zapsplat.com/) - 免费音效资源
- [Pixabay](https://pixabay.com/sound-effects/) - 免版权音效

## 临时方案

如果暂时没有音效文件，可以：
1. 在代码中注释掉音效播放部分
2. 或使用在线音效 URL（需注意跨域问题）
